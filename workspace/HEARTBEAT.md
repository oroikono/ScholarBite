# ScholarBite Heartbeat

## Daily arXiv Report

Check the current time. If it is between 07:00 and 08:00 UTC and you have NOT already sent today's report, perform the following:

1. Use the `arxiv-researcher` skill to fetch papers from the last 24 hours
2. Read the user's research profile from USER.md
3. Score and rank papers by relevance
4. Format a daily digest with the top papers
5. Send the report to the active channel

If it is not report time, or the report has already been sent today, do nothing.

## Paper Memory

After sending a report, save the list of paper IDs and titles to memory so you can reference them in follow-up conversations.
